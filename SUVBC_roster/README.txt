SUVBC roster plugin 

Installation:
1) Once you have successfully downloaded SUVBC_roster.zip, place it somewhere easy to find, such as your desktop.

2) On your WordPress dashboard, navigate to Plugins > Add New > Upload. (If you are having trouble finding Upload, it is located underneath the Install Plugins title.)

3) Now click Browse... or Choose File and find the SUVBC_roster.zip file. Once you have selected SUVBC_roster.zip, click Open, then Install Now. The following read-out should appear to ensure that successful installation occurred: 
“Unpacking the package… 
Installing the plugin… 
Plugin installed successfully.”

4) Click Activate Plugin. After doing so, you will see “Plugin activated” in a box at the top of your screen. This plugin has now been successfully installed and is ready to be used. Continue to the set-up below to lean how to properly add and detelete players.

	NOTE: If you are using localhost, just extract the SUVBC_roster.zip into your	www/wordpress/wp-content/plugins and then activate it in your admin plugins page.

---------------------------------------------------------------------------------------------------
SET-UP
1) On the dashboard you will find a tab that says Roster. Click this then add new

2) The title of this post does not matter. Enter [SUVBC_Roster] as the content and then publish.

---------------------------------------------------------------------------------------------------
ADD PLAYERS
1) On the bottom of the dashboard, click SUVBC Roster

2) Here you will be able to add, edit and remove players from this table. Enter in new players number, name, position, year and hometown.

3) For images, click on search image. A wordpress media uploader will appire. Select the right image or upload a new file

4) Once you have selected this image, click save image. The image soure will be filled out.

5) Enter in the players bio and then click save.

6) If eveything was done correcty, a player add message will show and you will see the name shown under "Added Players"

---------------------------------------------------------------------------------------------------
EDIT PLAYERS
1) Select player that you want ot edit by clicking on it in the players table. It will become hightlighted 

2) Edit any information that is needed then click UPDATE

3) A response message will apiear and the player's information will be changed on the table

---------------------------------------------------------------------------------------------------
DELETE PLAYERS
1) Select the player you wish to remove from the table by clicking on their name. It will become highlighted  

2) Click Delete

3) Press ok if you want to delete this player. You will now see the player is removed from the Added Players table
